# HS2 Software Libraries

This directory contains the F Prime framework as a submodule.

## Setting Up F Prime

To initialize and update the F Prime submodule:

```bash
# Initialize the fprime submodule
git submodule add https://github.com/nasa/fprime.git lib/fprime

# Or if cloning the repository for the first time:
git submodule update --init --recursive
```

## F Prime Version

The project uses F Prime as a framework. The specific version is tracked via git submodule.
